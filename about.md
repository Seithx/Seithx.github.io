# About Me

## Asaf Lecht | אסף לכט

Full-Stack AI/Data Engineer based in Israel.

I build production-ready systems that replace manual workflows with AI-powered automation. My work spans manufacturing video analysis, document processing pipelines, RAG knowledge bases, browser automation, and Hebrew-first developer tools.

This blog is my learning journal -- an honest record of building software in the real world. I write about the struggles, breakthroughs, scope creep, and lessons learned along the way.

---

## What I Work On

- **AI-Powered Automation** -- Invoice processing with Gemini, video analysis for manufacturing, AI-generated daily digests from messaging groups and newsletters
- **Browser Automation** -- Persistent headless browser patterns for wrapping web applications that lack APIs
- **RAG Pipelines** -- Cleaning and structuring document collections for AI knowledge bases, with strict quality gates to prevent LLM hallucination
- **Hebrew Developer Tools** -- [Smart Hebrew RTL](https://chromewebstore.google.com/detail/mceaejlgpljlmfmejpgfjmpolkehippf), a Chrome extension with 80+ users that fixes Hebrew typing direction in AI chat interfaces like ChatGPT, Claude, and Gemini
- **Desktop & Mobile Apps** -- Shipping calculators (React + Electron), audio/video converters, digital signage systems

## Tech Stack

Python, TypeScript, Google Cloud (Cloud Run, Apps Script), Gemini AI, FastAPI, Flask, React, Electron, Playwright, Docker, PyQt6, ChromaDB, GitHub Actions

## על קצה המזלג (בעברית)

שמי אסף לכט, מהנדס תוכנה ונתונים עם התמחות בבינה מלאכותית ואוטומציה. אני מפתח מערכות AI לייצור, עיבוד מסמכים, צינורות RAG, ואוטומציה של דפדפן. גר בישראל ועובד עם טכנולוגיות כמו Python, TypeScript, Google Cloud, ו-Gemini AI.

הבלוג הזה הוא יומן הלמידה שלי -- תיעוד כנה של תהליך הפיתוח, האתגרים, הפריצות דרך, והלקחים שלמדתי בדרך.

פיתחתי את [Smart Hebrew RTL](https://chromewebstore.google.com/detail/mceaejlgpljlmfmejpgfjmpolkehippf) -- תוסף לכרום שמתקן את כיוון הכתיבה בעברית בצ'אטים של AI כמו ChatGPT, Claude ו-Gemini. לתוסף יש מעל 80 משתמשים פעילים בחנות של Chrome.

אני גם תומך בפודקאסט [על המשמעות](https://mashma.net) -- שיח אינטלקטואלי מוביל בישראל.
